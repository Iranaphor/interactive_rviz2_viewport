# RViZ Interactive Viewport Control

This package is focused around developing interactive and experimental rviz visuals as examples and samples to be incorporated into future projects.

## Launch

To run the system, be on ubuntu and open x-host for docker to use `xhost +local:docker`. Then cd into the `docker/` directory and `docker compose up`.

Click on the interactive markers to move between them. Click on one a second time to rotate around.
